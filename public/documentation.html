Documentation Page

In this project we will be setting up a movie api. We will be returning a list of all movies to the user. This will include a description as well as genre, movie poster, and even information about the director. After which users will be allowed to register an account to keep tabs on their favorite movies and movies they want to watch.

Request: Return a list of ALL movies to the user

URL: '/movies'
Method: GET
Response: JSON the movie titles list

Request: Return movie data (description, genre, director, image URL)

URL: '/movies/:movieInformation'
Method: GET
Response: JSON the movie information of selected movie

Request: Return movies categorized by genre

URL: '/movies/:genre'
Method: GET
Response: JSON listings of movies by genre

Request: Return data about a director

URL: '/directors/:directorInfo'
Method: GET
Response: JSON list of directors, followed by information on each director

Request: allow new users to register

URL: '/users'
Method: POST
Response: JSON required user information and submitted to database

Request: allow users to update their user info(username)

URL: '/users/:accountInfo/'
Method: PUT
Response: JSON users information that can be updated
